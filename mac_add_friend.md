# 카카오톡 친구추가 자동화 (macOS)

`kakao_mac_add_friend.py` 는 카카오톡 macOS 앱에서 친구추가 자동화를 시작하기 위한
별도 진입 파일입니다.

현재는 연락처 방식의 친구추가 플로우를 실행합니다. 이름/전화번호를 입력하고
`친구 추가` 버튼을 확정한 뒤, 팝오버가 닫히면 성공으로 보고 오류 문구가 남으면
실패로 판정합니다.

## 스텝

- [x] Step 1 — 접근성 권한 확인 후 이미 실행 중인 카카오톡을 frontmost 로
- [x] Step 2 — `⌘+1` 로 친구 탭 포커스
- [x] Step 3 — 친구추가 팝오버 열기 (`AXPress`, 마우스 좌표 클릭 없음)
- [x] Step 4 — 연락처 탭에서 이름 / 전화번호 입력
- [x] Step 5 — `친구 추가` 확정 후 팝오버 닫힘 / 오류 문구로 결과 판정

## 요구사항

- macOS, Python 3.8+
- 카카오톡 데스크톱 앱을 미리 실행
- 터미널/IDE 또는 Python 실행 파일에 손쉬운 사용(Accessibility) 권한 부여
- 외부 라이브러리 없음 (`kakao_mac.py` 의 기존 `osascript`, `ctypes` 기반 로직 재사용)

## 사용법

```bash
python3 kakao_mac_add_friend.py "테스트1" "010-1111-1111"
```

인자를 생략하면 모든 step 시작 전에 이름과 전화번호를 입력받습니다.

성공 시 출력:

```text
[Step 1] 카카오톡을 포커스했습니다.
[Step 2] 친구 탭으로 포커스했습니다. (⌘+1)
[Step 3] 친구추가 팝오버를 열었습니다. (AXPress)
[친구추가] 친구추가 팝오버 진입 준비를 완료했습니다.
[Step 4] 이름/전화번호 입력 완료: name='테스트1', phone='01011111111', 친구 추가 버튼=활성
[Step 5] 친구 추가 완료: 팝오버가 닫혔습니다.
```

## 구현 방식

- 접근성 권한 확인은 기존 `kakao_mac.py` 의 `ensure_accessibility_or_exit()` 를 그대로 사용합니다.
- 카카오톡 실행 확인과 frontmost 보장은 기존 `step1_activate_kakao()` 를 그대로 사용합니다.
- 친구 탭 이동은 기존 `step2_focus_tab("1")` 을 감싼 `step2_focus_friends_tab()` 으로 처리합니다.
- 친구추가 진입은 친구 탭 header 의 36x36 아이콘 버튼 중 가장 오른쪽(사람+ 아이콘)을
  접근성 트리에서 찾아 `AXPress` 합니다. 마우스 좌표 이동/클릭은 사용하지 않습니다.
- 성공 판정은 팝오버 내부의 `친구 추가`, `연락처`, `ID` 요소가 잡히는지 확인합니다.
  이미 열려 있으면 다시 누르지 않고 성공으로 처리합니다.
- 연락처 입력은 `set value` 대신 각 text field 에 AX focus 를 준 뒤
  `⌘A` / Delete / 클립보드 붙여넣기를 사용합니다. `set value` 는 화면 값만 바꾸고
  카카오톡 내부 validation 이벤트를 발생시키지 않아 버튼이 비활성으로 남는 것이
  실측되었습니다.
- 확정은 `친구 추가` 버튼의 `AXPress` 액션으로 수행합니다. 이후 팝오버가 닫히면
  성공, 팝오버가 유지되고 `할 수 없습니다` 같은 오류 문구가 보이면 실패로 판정합니다.
- 카카오톡이 실행되어 있지 않거나 frontmost 전환에 실패하면 `KakaoMacError` 로 종료합니다.

## 단축키 근거

카카오 고객센터의 PC/태블릿/워치 이용 카테고리 내 `Mac 버전 단축키가 궁금해요.`
항목에서 macOS 카카오톡 단축키가 다음처럼 안내됩니다.

- 친구 탭 열기: `⌘ + 1`
- 채팅 탭 열기: `⌘ + 2`
- 더보기 탭 열기: `⌘ + 3`

참고 URL: <https://cs.kakao.com/helps?service=8&category=1056&locale=ko>

## 테스트 결과

실측 테스트:

```bash
python3 kakao_mac_add_friend.py "테스트1" "010-1111-1111"
```

결과:

```text
[Step 1] 카카오톡을 포커스했습니다.
[Step 2] 친구 탭으로 포커스했습니다. (⌘+1)
[Step 3] 친구추가 팝오버를 열었습니다. (AXPress)
[친구추가] 친구추가 팝오버 진입 준비를 완료했습니다.
[Step 4] 이름/전화번호 입력 완료: name='테스트1', phone='01011111111', 친구 추가 버튼=활성
[Step 5] 친구 추가 실패: 테스트1 | +82 | 01011111111 | 친구 추가 | 입력하신 번호를 친구로 추가할 수 없습니다. | 친구 추가 | 연락처 | ID
```

카카오톡이 `010-1111-1111` 을 `01011111111` 로 정규화했고, 버튼은 활성화됐지만
최종 확정 후 `입력하신 번호를 친구로 추가할 수 없습니다.` 오류 문구를 반환했습니다.
